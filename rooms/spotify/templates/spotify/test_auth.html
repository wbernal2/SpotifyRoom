Troubleshooting Steps

  1. Make sure your Spotify Developer Dashboard has the correct redirect URI: http://127.0.0.1:8000/spotify/redirect
  2. Check that your Client ID and Client Secret are correct
  3. After clicking authenticate, you'll be redirected to Spotify and then back to this page
  4. Watch the server console logs during this process for detailed information
  5. If authentication still fails, try clicking "Create Test Token" to manually create a token for your session
